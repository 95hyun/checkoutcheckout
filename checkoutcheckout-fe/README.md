# CheckoutCheckout 프론트엔드

공부 기록 서비스의 프론트엔드 프로젝트입니다.

## 기술 스택

- React
- TypeScript
- Zustand (상태 관리)
- Tailwind CSS (스타일링)
- Axios (API 통신)
- React Router (라우팅)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 프로젝트 구조

- `/src/api`: API 통신 관련 코드
- `/src/components`: 재사용 가능한 UI 컴포넌트
- `/src/hooks`: 커스텀 React 훅
- `/src/pages`: 라우트별 페이지 컴포넌트
- `/src/store`: Zustand 상태 관리
- `/src/types`: 타입 정의
- `/src/utils`: 유틸리티 함수
